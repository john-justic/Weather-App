# JOHN JUSTICE WEATHER DASHBOARD
## Description
This program provides  weather conditions, relative humidity, wind speed and direction, in addition to five days in future weather forecast

This project has been built by:

- bootstrap files
- HTML
- CSS
- Weather API
- Logo.svg
